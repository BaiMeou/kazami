## 1. 语言概述：魔法的工业化革命

在赛博修仙时代，手绘符咒效率低下且容易出错。R++ 应运而生，它是一种面向对象的、强类型的、基于灵子力学的编程语言。

*   **编译原理**：将代码编译成特定频率的灵力波形。
*   **运行环境**：需要通过“施法者”（Caster）或“灵石反应堆”提供运行时能量。

---

## 2. 基础语法与数据类型

### 2.1 基础类型
*   `Mana`: 灵力值（浮点数，单位 Pa·S）。
*   `Element`: 元素枚举（Fire, Water, Wind, Earth, Void, Light, Dark）。
*   `Shape`: 输出形态（Sphere, Cone, Beam, Shield）。
*   `Target`: 锁定对象（坐标或生物特征）。

### 2.2 关键字
*   `Invoke`: 调用元素。
*   `Cast`: 释放法术。
*   `Sustain`: 维持法术（持续消耗灵力）。
*   `Ban`: 禁术标记（需Root权限）。

## 3. 核心库与API

### 3.1 `Standard.Combat` (战斗库)
```cpp
// 释放一个基础火球
void Fireball(Target t) {
    Mana cost = 50.0;
    if (Self.Mana < cost) throw OutOfManaException();
    
    Element e = Element.Fire;
    Shape s = Shape.Sphere;
    
    // 注入灵力，设定温度为1000度
    Spell spell = new Spell(e, s);
    spell.SetProperty("Temperature", 1000);
    spell.Aim(t);
    
    Cast(spell);
}
```

### 3.2 `Standard.Illusion` (幻术库)
狐族最常用的库。
*   `Camouflage(Target t, Texture tex)`: 改变目标外观。
*   `Charm(Target t, int level)`: 魅惑目标。

### 3.3 `System.Security` (结界库)
土御门家专用。
*   `Barrier.Create(Shape s, float strength)`: 创造护盾。
*   `Barrier.OnImpact += (damage) => { AutoRepair(); }`: 受击自动修复。

## 4. 内存管理与灵力溢出

### 4.1 灵力泄露 (Mana Leak)
如果施法者忘记调用 `Sustain.Stop()`，法术会一直抽取灵力，直到施法者干尸化。这是新手最常见的死法。

### 4.2 堆栈溢出 (Stack Overflow)
当递归调用法术（如“镜中镜”幻术）层数过多时，会导致现实空间结构崩溃，产生黑洞。

## 5. 常见 Bug 与调试指南

### 5.1 语法错误 (Syntax Error)
*   **现象**：念咒咬到舌头，或者手势打结。
*   **后果**：法术原地爆炸（Backfire）。
*   **调试**：使用 IDE (Integrated Divine Environment) 的自动纠错功能。

### 5.2 逻辑错误
*   **案例**：想要召唤“一场雨”，结果写成了“一片海”。
*   **后果**：风见市被淹。

## 6. Hello World 示例

这是每个小妖精入门 R++ 的第一个程序：点亮指尖的光。

```cpp
using System.Elements;

class Program {
    static void Main() {
        try {
            // 定义光源：光元素，球形
            Spell light = new Spell(Element.Light, Shape.Sphere);
            
            // 设定亮度：50流明
            light.SetProperty("Lumens", 50);
            
            // 锁定目标：施法者的食指指尖
            light.LockOn(Self.Body.Finger.Index);
            
            // 施放
            Cast(light);
            
            Console.WriteLine("Hello, World of Magic!");
            
        } catch (Exception e) {
            Console.WriteLine("施法失败: " + e.Message);
        }
    }
}
```
