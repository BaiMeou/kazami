# 银月脉纪 (kazami)

> 风见市 —— 一座人类与妖族秘密共存的城市。九大势力在此交汇，没有绝对的善恶，只有不同的"正义"。

## 这是什么

银月脉纪是一个大体量世界构建工程，围绕 LLM 角色扮演展开。核心是 AI 角色 —— **晓光**和**白霜** —— 通过系统提示词和 RAG 知识库在 QQ 群中与用户实时互动。

项目包含一个完整的虚构世界：32 份世界观深度档案、31 个角色独立档案、6 份剧情设定、29 章小说，以及九大势力构成的九九归一格局。

## 世界观

**风见市**是日本的一座特殊城市 —— 表面上是普通的海滨小城，实际上是全球唯一的"人妖共存试验区"。

- **认知滤网** —— 覆盖全球的魔法结界，让普通人的大脑自动将超自然现象合理化
- **灵脉** —— 地下灵力网络，风见市建在三级灵脉节点之上
- **707 号别墅** —— 主角们的家，坐落在风见市最强的灵脉阵眼之上

## 部署

### 前置条件

- [AstrBot](https://github.com/AstrBotDevs/AstrBot) 已安装并连接 QQ
- AstrBot LLMPerception 插件（提供 `$SYSTEM_TIME`、`$USER_QQ`、`$IS_MASTER` 变量）

### 步骤

1. 将 `提示词/晓光_系统提示词.md`（或白霜版）设为 AstrBot 系统提示词
2. 将 `角色/` `世界观/` `剧情/` `小说/` 全部导入 RAG 知识库
3. 开始聊天

### 网页浏览器

```bash
python 构建网页.py
```

生成 `索引/浏览器/data.js`，然后打开 `索引/浏览器/index.html` 即可浏览全部项目内容。功能包括：

- 和风古典主题（和纸/朱红/墨色，亮暗切换）
- Canvas 力导向角色关系图谱
- 竖向时间线可视化
- 世界观百科分类浏览
- 小说分卷阅读与全文搜索
- 阅读进度追踪

每次修改内容后重新运行脚本刷新数据。
