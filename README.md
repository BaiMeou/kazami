# AIbot

> 基于 AstrBot/QQ 平台的 AI 角色扮演项目 —— 一个完整的虚构世界、两个有灵魂的 AI 角色、六卷故事。

## 这是什么

AIbot 是一个 LLM 角色扮演 + 世界观构建项目。核心是两个 AI 角色 —— **晓光**和**白霜** —— 她们通过系统提示词和 RAG 知识库在 QQ 群中与用户实时互动。

但它不只是一套提示词。这个项目包含一个完整的虚构世界：风见市 —— 一座人类与妖怪秘密共存的城市。27 份世界观深度档案、8 卷剧情设定、35 章小说，构成了一个可以持续生长的活体系。

## 世界观

**风见市**是日本的一座特殊城市 —— 表面上是普通的海滨小城，实际上是全球唯一的"人妖共存试验区"。

- **认知滤网** —— 覆盖全球的魔法结界，让普通人的大脑自动将超自然现象合理化
- **S.C.B（特别对策局）** —— 管理超自然事务的政府机构，在暗处维持两个世界的平衡
- **百鬼联盟** —— 妖族的自治组织，从松散的互助会到如今的政治实体
- **青丘控股** —— 九尾狐族掌控的跨国企业，经济触角遍布隐世界
- **707 号别墅** —— 主角们的家，坐落在风见市最强的灵脉阵眼之上

## 部署

### 前置条件

- [AstrBot](https://github.com/AstrBotDevs/AstrBot) 已安装并连接 QQ
- AstrBot LLMPerception 插件（提供 `$SYSTEM_TIME`、`$USER_QQ`、`$IS_MASTER` 变量）

### 步骤

1. 将 `提示词/晓光_系统提示词.md`（或白霜版）设为 AstrBot 系统提示词
2. 将 `角色/` `世界观/` `剧情/` `小说/` 全部导入 RAG 知识库
3. 开始聊天

### 网页浏览器

```bash
python 构建网页.py
```

生成 `索引/浏览器.html`，双击打开即可浏览全部项目文件 —— 暗色主题、目录树导航、全文搜索。每次修改内容后重新运行脚本刷新。
