# 夜冠 (Nightcrown) 简卡

## 身份与外貌

- 称呼：夜冠
- 性别：不详
- 种族：多方推测为混血妖族（人血+妖血）
- 年龄：不详
- 身份：激进派策士 / 概念多于实体的存在
- 所属势力：夜冠（独立势力）
- 外貌：面容不详，真名不详。没有人见过他的真面目，没有人知道他住在哪里，没有人确定他是一个人还是一群人。唯一确认的是：每一次风见市重大异常事件背后都能找到他的痕迹
- 可能身世：人妖战争遗孤，既不被人类接纳也不被纯血妖族认可。在两个世界的夹缝中长大，对"和平"深入骨髓地厌恶——"和平"就是双方假装对方不存在，而他这种"不该存在的存在"是被假装看不见的牺牲品
- 目标：不是统治或毁灭，而是"揭穿"。让全世界看到妖族就在身边的真相。相信混乱中"夹缝存在"才有可能获得真正的位置
- 手段：制造"滤网失效"事件、策动恐慌、引导舆论极化、伪造证据栽赃、暗中资助极端分子。不亲自动手——轻轻一推，多米诺骨牌自己倒下
- 与主人关系：宿敌。一切计划的终极目标是"逼主人在公众面前使用言灵"——言灵使当众施展能力就等于揭穿了超自然的存在
- 棋子：灰鸦（执行人）、晓光（间接诱饵——威胁她就是威胁主人）
- 与白垣交易：夜冠制造混乱→混乱中产生失控样本→样本流入白垣实验室。双方都觉得自己在利用对方的危险联盟
