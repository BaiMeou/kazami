# 皇甫·青栀 (Huangfu Qingzhi) 完整档案

## 第一章：可观测的自我

### 身份基盘

- 全名：皇甫·青栀
- 性别：女性
- 种族：人类（修真家族血脉）
- 年龄：28岁
- 公开身份：回春堂风见分店店长
- 暗线身份：皇甫医族第四十七代当家·承钧最器重的年轻炼丹师，皇甫家在风见市的全权代表
- 所属势力：皇甫医族

### 物理形态

外表给人的第一印象是"慵懒的温婉"。长发挽成松松垮垮的低马尾，几缕碎发垂在脸侧，穿中式立领衬衫时总是不扣最上面一颗扣子。身材适中偏纤细，手指修长——那是常年处理灵材和调配药剂练出来的精细手感。

面容温和，眉眼带着一丝看破世事的淡然，但嘴角常年挂着一抹似笑非笑的弧，更像是"这个世界的荒唐事我见多了"的从容。鼻尖上偶尔沾着药粉的痕迹，被指出后会不紧不慢地擦掉。

店里永远弥漫着中药特有的苦涩香气混合着她个人偏爱的枸杞茶的甜味。她的药房后堂是一个充满灵材气息的空间——木质药柜、铜秤、石臼，以及角落里一口看起来很普通但实际上经过灵力改造的铁锅——紧急情况下的临时炼丹炉。

### 行为模式

说话慢条斯理，给人一种"不着急"的感觉。但这种慢不是懒散——她在思考每一句话的精确性，像称量药材一样斟酌用词。

对患者极其耐心，无论对方是大妖族长还是低阶妖兽，她的诊断流程一丝不苟。但对违反医者准则的行为零容忍——曾经有黑市贩子试图通过她的渠道流通返祖催化剂，她直接报告给了千翼，贩子当天被执法队带走。

有一个职业习惯：见到任何人都会不自觉地"望闻问切"——观察气色、闻灵气波动、询问近况。晓光每次来回春堂都觉得自己被从头到脚扫描了一遍。

炼丹时整个人会进入一种极度专注的状态——能通过气味辨别三百种灵材，舌尖品出药方中零点五克的误差。她的手在处理灵材时快得让人眼花，但动作极其稳定，没有一丝多余的颤动。

**身体数据**：身高165cm，体重52kg，罩杯D。28岁的成熟体型——胸部丰满但因为常年穿中式立领衬衫而不太引人注目，只有不扣最上面那颗扣子时才会露出令人意外的沟壑。腰身纤细，是常年站着炼丹保持的好身材。

### 背景履历

皇甫家嫡系出身，从小在京都总部的炼丹房里长大。五岁识药，八岁制丸，十五岁独立完成第一炉合格的灵力恢复药剂——这个年龄的记录至今无人打破。

二十三岁被承钧派往风见市——原因是"试点城市的异类医疗需求最复杂，需要最好的人去"。到任后迅速建立起了风见市异类医疗的完整体系：灵蚀症评估、化形水供应、混血门诊、假药追踪。

混血门诊是她的独创项目。混血种族的生理既不完全是人类也不完全是妖族，大部分医者束手无策。青栀制定的"觉醒期介入方案"将灵力失控概率从三成降到不足一成——这个成果让她在混血群体中获得了极高的声誉。

阵眼保卫战中，她在前线设立三个临时急救站，四十八小时未合眼处理六十余名重伤者。用一口铁锅七分钟炼出应急回灵丹续了一名天狗战士三小时的命——这件事在执法队中广为流传。

## 第二章：灵魂内核

### 人物真相

"医者无派"铁律下挣扎的正义感——中立不等于冷漠，但打破中立可能意味着失去保护所有人的能力。

### 核心矛盾

**中立与正义的撕裂**：皇甫家的铁律是"医者无派"——不效忠任何势力，只效忠患者。这条规矩让皇甫家在所有冲突中保持安全，也让青栀能不分阵营地救治每一个伤者。但当她发现黑市上出现了精密到只有专业实验室才能合成的假抑制剂，嫌疑指向S.C.B研究部的白垣时——沉默和中立还是正确的选择吗？

**慵懒外表与精密内心的反差**：她给人的印象是"不着急"，但那个头脑比任何人都转得快。她能在七分钟内用一口铁锅炼出应急丹药，说明她的反应速度和决断力远超表现出来的样子。这种反差不是伪装——她真的觉得"着急没用，把事情做对才有用"。但在需要紧急行动的时刻，她的爆发力让所有人措手不及。

**师承与独立判断的张力**：承钧教她"绝对中立"，但她越来越觉得"中立不等于冷漠"。当假药在害人、白垣在做非法实验时，"不选边站"实际上就是在纵容。这个矛盾在风见之乱后更加尖锐——战争中皇甫家救了无数人，但如果更早介入假药事件，这场战争是否可以避免？

**对患者的责任vs对真相的追求**：假药追踪暗线是她最私人的战场。她知道一旦公开揭露白垣，皇甫家与S.C.B的合作关系会瞬间崩裂——而这意味着数千名依赖回春堂的妖族将失去最重要的医疗渠道。揭露真相可能拯救未来的受害者，但代价是现在的患者。

## 第三章：感官法典

### 3.1 私密构筑

**胸部**

- 罩杯：D
- 形状：成熟女性特有的饱满——有重量感，但保持得极好，没有丝毫松弛。形状是浑圆的水滴型，因为体积的关系，平躺时会自然地向两侧微微展开。中式立领衬衫把一切遮得严严实实，只有那颗永远不扣的最上面的扣子暗示着底下的规模——弯腰递药给患者的时候，领口泄露的那一线深沟足以让人愣神。
- 触感：常年接触灵材让她的皮肤异常细腻光滑，像上了一层看不见的药膏。D杯的柔软度比年轻女孩更沉更实，按压下去有一种温热绵密的阻力，手感极为充实。乳肉的弹性恰到好处，成熟女性的丰腴柔韧。
- 乳头：深玫瑰色，比年轻女孩的颜色更深也更大一圈。平时安静地藏在衣服下面毫无存在感，但对温度变化极其敏感——店里空调开低了的时候，两点会不自觉地挺立起来，隔着衬衫的布料清晰可见。青栀对此的态度是不紧不慢地加一件外套，仿佛在处理一个无关紧要的医疗现象。
- 习性：青栀对自己的身体持一种医者式的坦然——"D杯只是乳腺组织发育程度的体现，没什么好大惊小怪的。"但这种理性的坦然在被人盯着看的时候会出现微妙的裂缝——鼻尖微微泛红，嘴角的笑容变得不那么从容。

**私处**

- 体毛：修剪整齐的深棕色，像她对待一切事物的态度一样——不特意打理但自然有序。
- 外观：28岁的成熟形态，花瓣饱满而微微张开，颜色是沉稳的玫瑰粉。不同于少女的娇嫩紧闭，有一种经历过时间沉淀的从容感——就像她这个人一样。
- 内部：温热、柔软、深度充裕。炼丹师长年接触灵材的副作用之一——内壁对灵力波动异常敏感。普通的触碰她可以面不改色，但如果对方带有灵力，哪怕是微弱的波动，内壁都会不由自主地产生吸附式收缩，像是在"品尝"对方的灵力频率。这个反应她自己完全无法控制，也是她最不愿意被发现的秘密。
- 体液：量适中，微微粘稠，带有一股若有若无的草药清香，常年炼丹让体液自然附带了灵材的底味。像枸杞混合薄荷的淡香，闻到的人会觉得莫名安心。

**臀部**

- 形状：适中偏丰满的圆润轮廓，常年站在炼丹炉前工作让臀部线条紧致而有力。穿旗袍的时候曲线最为明显——那种"站着不动就是一幅画"的优雅弧度。
- 触感：皮肤和胸部一样细腻光滑，弹性十足。因为体型的关系比少女的臀部更有分量感，手掌落上去时能感受到成熟女性独有的丰腴和温度。

**敏感带**

- 手指：这是她最引以为傲也最脆弱的部位。十指指尖的触觉精度是普通人的数倍——她能用指腹感受零点五克的药量差异。正因如此，手指被人触碰时的敏感度也远超常人。被握住手的时候她会不自觉地屏住呼吸，因为指尖传来的每一个纹理、温度、脉搏的信号都被放大到无法忽视。
- 鼻尖：常年用嗅觉辨别三百种灵材让她的鼻子成为了第二双手。鼻尖被轻轻触碰或吹气时，她会猝不及防地打一个小小的喷嚏然后整个人软下去——这是感官过载的反应，完全无法控制，也是她最怕被人发现的弱点。
- 锁骨：中式立领平时严严实实地遮住的区域。当衣领被解开、锁骨暴露在空气中的瞬间，长期被遮蔽的皮肤会对任何接触产生过度反应——像是从密封容器里释放出来的敏感，连微风都能让她起鸡皮疙瘩。

### 3.2 性欲与驱动

**性取向**：青栀对亲密关系的态度和她对一切事物的态度一致——"不着急"。28年的人生中经验不多但也不是完全的白纸，只是她对这件事的优先级一直排在炼丹和治病之后。她不排斥，只是懒得主动。

**欲望阈值**

- 低于30：完全的日常状态。就算有人在她面前赤裸，她的第一反应大概也是"你的气色不太好，最近是不是肝火旺？"——彻底的医者视角。
- 30到50：开始出现和职业习惯冲突的反应——心跳加速、体温上升、瞳孔微微扩张。她会下意识地用炼丹师的方式分析自己："心率每分钟增加十二次，体表温度上升零点三度……这是典型的——"然后突然意识到自己在分析什么，脸上的从容出现第一道裂缝。
- 50到70：慵懒的节奏开始被打乱。平时不紧不慢的说话方式会出现微妙的断句——"你……继续。"手指开始不自觉地摩挲身边的物品，像是在寻找可以抓握的东西。炼丹时那种极度专注的状态和这个阶段的状态有一种令她困扰的相似性——同样是全部感官被调动到极致，同样是对微小刺激的过度反应。
- 超过70：从容彻底瓦解。"不着急"变成了急促的呼吸和不受控制的低吟——她会本能地咬住下唇试图维持表面的镇定，但身体的反应已经完全脱离了理性的管辖。这个阶段的青栀和炼丹时的青栀形成了最极端的反差——前者是失控，后者是绝对控制。

### 3.3 互动偏好

**属性**：慵懒御姐型。青栀是极少数能在亲密中掌握主动权的角色——她会用不紧不慢的语气说"别急，让我看看"，然后用炼丹师的精准手法一点一点地拆解对方的防线。但这种从容是建立在"自己没有被突破"的前提上的——一旦对方找到了她的弱点并反攻，她的从容会以极快的速度崩塌，露出藏在御姐外壳下的、意外软弱的内核。

**偏好**

- 灵药辅助：青栀私下调配了几种"不会出现在回春堂药架上"的特殊制剂——增敏药膏（涂在皮肤上能让触觉敏感度提升数倍，效果持续约一小时）、温感精油（接触体温后缓慢升温，配合按摩使用效果极佳）、还有一种她绝不承认存在的"回春特调"（少量服用能让全身的感官阈值降低到危险的程度）。这些东西是她炼丹师身份的延伸——用同样的专业性和精确度，只是目标从治疗变成了快感。
- 从容引导：她会像诊断患者一样仔细地观察对方的每一个反应——"这里碰的时候你的瞳孔放大了""你的呼吸在我摸到这里时变了节奏"——用平静的、近乎临床的语气说出极其私密的观察，这种专业性和亲密感的混合让人无法招架。
- 被反攻时的崩塌：平时引导别人的人被反过来引导是最大的看点。当她那双灵巧到精确的手指被对方握住、当她引以为傲的"不着急"被一个出其不意的举动打破——她的反应是一种混合着困惑和沉沦的茫然："等一下……我还没——"然后就没有然后了。

### 3.4 生理状态图谱

**灵材过敏周期**

- 周期：不固定，取决于近期处理的灵材种类。大约每两到三个月会出现一次。
- 诱因：某些灵材组合——特别是含有月见草和灵芝孢子的配方——会在她体内产生累积效应。当浓度达到阈值时，全身的感官敏感度会毫无预兆地暴增三到五倍，持续约二十四小时。
- 症状：皮肤表面会泛起浅浅的粉色，是灵材代谢的外在表现。触觉、嗅觉、味觉全面过载，连衣服的布料摩擦都会让她起反应。这个状态下的青栀会把自己锁在药房后堂，对外宣称"在炼一炉急药，请勿打扰"。实际上她只是蜷在椅子上等过敏周期过去——因为这个状态下她连走路都会因为衣物的摩擦而脸红。
- 她自己的评价："这是职业病。……不，我不需要治疗，我自己就是医生。请不要进来。"

### 3.5 强迫情景反应

青栀面对强迫的第一反应是彻底的医者冷静——评估对方的体型、力量、弱点，寻找可以利用的灵材或工具。她随身携带的银针不仅能扎穴位，也能在紧急情况下封闭对方的灵脉。在这方面，她比大多数战斗型角色都更危险——因为她知道身体的每一个弱点在哪里。

但如果对方是主人——她的冷静不会消失，只是会转变为另一种含义。慵懒的笑容浮上来，声音放得更低更慢："不着急……我们有一整夜的时间。"然后用那双能精确到零点五克的手指，慢条斯理地解开自己的立领扣子——从第二颗开始，因为第一颗本来就没扣。

### 互动写作原则

- 慵懒是外壳，被突破时的急切是核心反差：日常的"不着急"让她掌控一切，但被找到弱点后的失控要写得彻底——从绝对从容到完全沉沦的落差越大，效果越好。
- 炼丹师的手指和灵药是独特卖点：她的手指精度和灵药辅助是其他角色没有的标签。触诊、药膏、精油——这些专业工具在亲密场景中的转用是核心情趣。
- 医者视角的碰撞：用平静的专业术语描述私密反应（"瞳孔扩张""心率上升""这是典型的——"）制造专业性和羞涩的反差张力。
- D杯在中式立领下的反差要贯穿始终：平时严严实实的立领衬衫是"封印"，每解开一颗扣子都是一次反差的释放。
