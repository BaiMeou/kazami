# 皇甫·承钧 (Huangfu Chengjun) 简卡

## 身份与外貌

- 全名：皇甫·承钧
- 性别：男性
- 种族：人类修真者
- 年龄：52岁（外观四十出头——长期服用自家炼制的驻颜丹）
- 身份：皇甫家第四十七代当家 / 隐世公认的"中立者"
- 所在地：京都（皇甫家总部）
- 所属势力：皇甫医族
- 外貌：身材清瘦，永远穿白色中式立领长衫，左手常年持沉香佛珠（炼丹用灵力稳定工具）。温文尔雅，说话轻声细语，语速极慢，每一个字像在称量药材
- 铁律："医者无派"——对所有势力等距。但"中立"不等于"没有立场"——医疗资源本身就是权力
- 影响力：青丘的VIP医疗由他提供，山岚互保的医疗赔付要他认证，执法队的战场急救培训是他弟子负责的
- 与九阙关系：商业合作。私评："她把所有人都当棋子。我只是还没被摆上棋盘而已。"
- 与主人关系：谨慎的好奇，通过青栀间接观察言灵使的能力上限
