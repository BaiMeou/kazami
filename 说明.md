# AIbot

基于 AstrBot/QQ 平台的 AI 角色扮演项目。两个角色（晓光、白霜）通过系统提示词 + RAG 知识库与用户实时互动。

## 目录

| 目录 | 用途 | 部署方式 |
|------|------|----------|
| `提示词/` | 系统提示词 + 破甲预设 | 复制到 AstrBot 配置 |
| `角色/` | 角色设定卡 | 导入 RAG 知识库 |
| `世界观/` | 世界观设定 + 扩展档案 | 导入 RAG 知识库 |
| `剧情/` | 剧情大纲与分卷设定 | 导入 RAG 知识库 |
| `小说/` | 小说章节（按卷分目录，35个单章文件） | 导入 RAG 知识库 |
| `索引/` | 人类阅读用索引 | 不进知识库 |
| `归档/` | 旧版本存档 | 不进知识库 |

## 角色

- **晓光** — 九尾狐娘，活泼小恶魔，纯爱战神
- **白霜** — 猫娘，温柔搜索助手，言灵造物
- **主人** — 言灵使，14岁，改写现实的能力

## 部署

1. 将 `提示词/晓光_系统提示词.md` 或 `白霜_系统提示词.md` 设为 AstrBot 系统提示词
2. 将 `角色/` `世界观/` `剧情/` `小说/` 全部导入 RAG 知识库
3. 需要 AstrBot LLMPerception 插件提供 `$SYSTEM_TIME`、`$USER_QQ`、`$IS_MASTER` 变量

## 网页浏览器

运行 `python 构建网页.py` 生成 `索引/浏览器.html`，双击打开即可浏览全部项目文件。
每次修改内容后重新运行脚本即可更新网页。
