# 银月脉纪 (kazami)

基于 AstrBot/QQ 平台的 AI 角色扮演 + 世界观构建项目。九大势力、31个角色、29章小说，构成一个可持续生长的虚构世界。

## 目录

| 目录 | 用途 | 部署方式 |
|------|------|----------|
| `提示词/` | 系统提示词 + 破甲预设 | 复制到 AstrBot 配置 |
| `角色/` | 角色设定卡 + 31个独立档案 | 导入 RAG 知识库 |
| `世界观/` | 世界观设定 + 32个扩展档案 | 导入 RAG 知识库 |
| `剧情/` | 剧情大纲与分卷设定 | 导入 RAG 知识库 |
| `小说/` | 小说章节（按卷分目录，29个单章文件） | 导入 RAG 知识库 |
| `索引/` | 人类阅读用索引与网页浏览器 | 不进知识库 |
| `归档/` | 旧版本存档 | 不进知识库 |

## 核心角色

- **晓光** — 九尾狐娘，活泼小恶魔，纯爱战神
- **白霜** — 猫娘，温柔搜索助手，言灵造物
- **主人** — 言灵使，14岁，改写现实的能力

## 九大势力

青丘控股 · 山岚商会 · 猫又文化联盟 · 百鬼联盟 · S.C.B · 夜冠 · 皇甫医族 · 土御门宗家 · 织网者

## 部署

1. 将 `提示词/晓光_系统提示词.md` 或 `白霜_系统提示词.md` 设为 AstrBot 系统提示词
2. 将 `角色/` `世界观/` `剧情/` `小说/` 全部导入 RAG 知识库
3. 需要 AstrBot LLMPerception 插件提供 `$SYSTEM_TIME`、`$USER_QQ`、`$IS_MASTER` 变量

## 网页浏览器

运行 `python 构建网页.py` 生成浏览器数据，打开 `索引/浏览器/index.html` 即可浏览全部项目文件。
