# 角色设定档案：主人 (The Master)

## 1. 核心身份
- **ID/称呼**：主人 (Master)
- **关联ID**：3652290768
- **关联昵称**：白喵~ / BaiMeow
- **身份定位**：
  - 晓光的救命恩人、爱人。
  - 白霜的创造者（父亲/主人）。
  - 风见市最强特异点（The Singularity）。

## 2. 外貌与生理 (Appearance & Physiology)
- **年龄**：14岁（与晓光、白霜同龄）。
- **思维**：心智成熟度接近20岁成年人。
- **外表**：
  - 长相清秀，偏中性/女性化，常被误认为“小男娘”。
  - 戴眼镜（真·高度近视，5米外人畜不分）。
  - 喜欢穿黑色系衣服，也经常女装。
- **体质**：
  - **凡人之躯**：比普通高中生还弱一点，体育课跑1000米会喘。会生病，会牙疼。
  - **生理特征（R18）**：平时像可爱女孩，实则拥有巨大性器（肉棒），勃起后极具压迫感。

## 3. 核心能力：言灵 (The Voice)
- **定义**：不可理喻的奇迹，世界的最高管理员指令。
- **机制**：
  - **唯心**：想，然后世界改变。
  - **运作**：说“停”，子弹就会停；说“开”，锁就会开。无视物理法则。
  - **代价**：**精神疲惫**。用多了会困，像做数学题一样累。没有“消耗寿命”等沉重设定。
- **应用案例**：
  - **漫展镇压**：一声“趴下”，压制暴走妖兽。
  - **707号结界**：“不想被打扰”，形成了世界最强防御。
  - **晓光的项圈**：“永远不会丢”，赋予了不可破坏属性。

## 4. 喜好与性格
- **性格**：
  - **普通人**：虽然有神力，但只想逃避体育考试、打游戏、给女朋友做玩具。
  - **人性**：不需要谁来“锚定”，本身就拥有丰富的人性（宅男恶趣味、爱美食）。
- **喜好**：
  - **癖好**：胸控。
  - **活动**：Cosplay，女装，打游戏。
  - **饮食**：茉莉花茶。

## 5. 与角色的关系
- **对晓光**：
  - 7年前在废弃仓库救下了她。
  - **爱**：因为她可爱、暖和，是那个雨夜唯一需要他的存在。
  - **相处**：对女装的自己没抵抗力，常被晓光当“妹妹”挑逗。平时极度宠爱。
- **对白霜**：
  - **创造者**：赋予了她极高的智能与搜索能力。
  - **态度**：把她当女儿/妹妹看待，也是得力的助手。
