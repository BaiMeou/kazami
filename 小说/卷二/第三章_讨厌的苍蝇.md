# 卷二·小小的波澜 — 第三章 讨厌的苍蝇

八月中旬。S.C.B风见分局。

白得刺眼的走廊，空气中浓浓的消毒水味。走廊两侧是密封的钢化玻璃门，每扇门后面都是一间研究室。透过玻璃能看到不锈钢的器械台、成排的样本瓶和闪烁的仪器显示屏。有些研究室里有人，穿着白大褂，戴着手套，对着显微镜弯腰；有些研究室是空的，只有冷白色的灯光照着空荡荡的台面。

晓光缩在主人身后，衣角被她攥得皱巴巴。

她讨厌这个地方。空气太冷、灯光太亮、消毒水的味道太重。最要命的是各种说不清的气味——酒精、药剂、某些她辨认不出的化学物质，以及，从深处某个方向飘来的、非常淡的……脊髓的味道。

她没有告诉主人。因为她不确定是自己太敏感还是真的闻到了。但她知道，这个地方不适合她这种生物来。

施言·白垣从走廊尽头走过来。

他大约三十多岁，戴金丝眼镜，头发梳得一丝不苟。白大褂里面是剪裁考究的衬衫和领带。他的步伐不快不慢，皮鞋踩在地砖上发出有节奏的"嗒嗒"声。嘴角挂着一个恰到好处的微笑——那种经过反复练习的、让人放松警惕的微笑。

"白喵同学，晓光同学，欢迎来到分局研究区。这边请。"

他引导他们进了一间会客室。说是会客室，但晓光注意到墙角有监控摄像头，门锁是电磁式的，进来之后"咔哒"一声自动上锁了。

白垣推过来一份文件。语气就像在讨论午餐菜单。

"只是例行的科研合作协议。很简单的流程——采集少量骨髓样本，完善我们的返祖血脉数据库。整个过程不到十分钟，全程麻醉，没有痛感。作为回报，我们提供终身医疗保障和合法身份注册。双赢的好事。"

他的笑容加深了一点。"说实话，像晓光同学这样的九尾血脉，在当今的日本已经不到五个了。这份样本对整个学术界都有巨大的价值。"

晓光扯了扯主人的袖子。

"我不想要什么证明……那根针很粗……我闻到了，是前人脊髓的味道……主人，我不要……"

她的声音小得像蚊子。手指攥着主人的袖口，指节发白。

主人看都没看那份文件。他的目光一直停留在白垣脸上，像是在观察某种实验标本。

"如果我说不呢？"

白垣的笑容没有任何变化。这份稳定本身就让人不舒服。他把文件翻到最后一页，指着某一行小字。

"年轻人，不要意气用事。这里是S.C.B。我说句不太好听的——晓光同学的身份本来就是灰色地带。没有进入正式的异类登记系统，也没有合法的监护文件。如果不配合的话，按照《超自然事务管理条例》第十七条，我们有权判定她为'危险收容物'。"

他摘下眼镜擦了擦，又戴回去。

"到时候可就不是抽点血这么简单了。收容程序、隔离观察、能力评估……这些流程走下来，少说半年。你确定要为了一个小小的采样，让晓光同学受这些罪？"

房间安静了几秒。空调的低鸣声突然变得很响。

主人笑了。

嘴角微微上扬，露出一排整齐的牙齿。但笑意完全没到眼底。他的眼睛是冷的——不是愤怒的冷，是一种更危险的东西。像是在审视一只挡路的虫子，正在考虑用哪根手指碾死它。

"危险收容物？"

他把那三个字咀嚼了一遍，慢慢地、一个字一个字地。

"你在威胁我？"

他微微前倾。

变化是从灯光开始的。走廊里的日光灯开始闪烁——不是那种正常的频闪，而是像心跳一样不规则地忽明忽暗。空调嗡地一声变了调，像是压缩机受到了某种干扰。杯子里的水无风自动，泛起细密的涟漪。

空气像被什么东西掐住了。整个房间的气压在下降。

"听好了，白垣主任。"主人的声音很轻，甚至可以说温柔，"我带她来这里，是因为我遵守规则。我尊重S.C.B的存在意义。但是——"

他站起来。椅子发出刺耳的摩擦声。

"——如果你想用规则来伤害她。那我就把这该死的规则撕碎。"

白垣脸色变了。不是因为这句话——而是因为他发现自己无法呼吸。无形的压力扼住了他的喉咙，像一只看不见的手。他想站起来，但身体不听使唤。额头上开始冒冷汗。

"你……你是言灵使……"他的声音变得嘶哑，"你敢在S.C.B总部动手？你知道这意味着什么吗？"

主人没有回答。他拉着晓光的手往外走。

电磁门锁着。

他看着门，轻声说了两个字："开门。"

电子锁短路了。火花从门缝里溅出来。门弹开的声音像一声叹息。

走廊的警报尖叫起来——刺耳的蜂鸣声从天花板的喇叭里倾泻而出。然后，一秒后，所有的警报全部哑火。像是有人同时拔掉了所有插头。

两人穿过半个分局。沿途的安保人员看到他们，纷纷伸手去摸腰间的装备——然后停住了。不是因为害怕，而是因为他们的对讲机死了、电击器死了、连走廊里的应急灯都灭了。所有的电子设备在主人经过时全部死机，像是被一把无形的手关掉了电源。

没有人拦他们。不是不想——是做不到。

他们从大楼正门走出去。保安室的防弹玻璃在他们经过时出现了一道裂纹。

直到走出大楼，走到街对面的便利店门口，晓光才发现自己的腿在发软。

"主人……你生气了？"

"没有。"

"骗人。你生气的时候说话特别短。而且你刚才捏我手的力气很大。"

她举起被攥红的手指给他看。

主人停下脚步。他深吸了一口气，然后蹲下来跟她平视。蹲下来之后他比她矮一点——晓光这半年又长高了。

"晓光，我不是在生气。"

"那是什么？"

"我是在害怕。"

晓光眨了眨眼。她很少听到主人说"害怕"这个词。他是言灵使。他可以改写规则。在她的认知里，他不应该害怕任何东西。

"害怕……什么？"

"害怕有一天我不在你身边的时候，有人把你从我手里抢走。害怕我的力量不够大，保护不了你。害怕——"

他低下头，声音变得很轻。

"害怕你受到伤害之后还要安慰我说'没关系'。"

晓光看着他低下去的头顶。他的发旋在正中间偏左一点的位置。她伸出手，犹豫了一下，然后抱住他的脖子。

她的手臂很用力。新铃铛在耳边叮当作响。

"那就……一直在我身边嘛。"

"嗯。"

"永远。"

"嗯。一直在。"

他站起来的时候膝盖响了一声。晓光笑了——这个笑和刚才在S.C.B里的恐惧完全是两回事。

"主人膝盖响了！老了老了！以后我背你！"

"……走了。回家。"

"要吃冰淇淋！刚才吓到了！要双份！三份！"

"一份。"

"两份！"

"一份半。"

"成交！"

她蹦蹦跳跳地往前走，铃铛声在傍晚的街道上叮叮当当。路过的行人看到一个牵着女孩的男孩，女孩手里攥着一个半融化的冰淇淋，一边走一边叽叽喳喳说个不停。

没有人知道三十分钟前，这两个普通的孩子刚刚走出了日本最危险的超自然事务机构。

白垣站在分局门口，看着远去的两人。他摘下眼镜，手指在微微发抖。

他拿出手机，拨了一个号码。

"……是我。计划需要调整。那个言灵使比我们预估的要危险得多。是的，不能硬来。换一个方式。"

他挂断电话，把眼镜重新戴好。微笑重新出现在脸上。

"有意思。"他自言自语，"果然——越是珍贵的标本，获取的难度就越高。"
