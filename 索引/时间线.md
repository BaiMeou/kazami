# 故事时间线

## 前史

| 时间 | 事件 |
|------|------|
| 14年前 | 晓光出生于北方山村，异色瞳+尾巴，被父母当"怪胎" |
| 8年前 (晓光6岁) | 雪灾断粮，父亲以两袋米+一桶油+500元将晓光卖给人贩子 |
| 8年前~7年前 | 晓光被关在地下室铁笼一年。脖子上被戴上刻着"晓光"的旧项圈 |
| 7年前 7月7日 | **命运之日**：7岁的主人发现废弃仓库中的铁笼，砸锁救出晓光。两人同天生日。"今天也是我的生日。" |
| 7年前~现在 | 707号别墅的七年羁绊。晓光从创伤中逐渐恢复 |

## 第一年（疗愈）
- 不敢上桌吃饭，噩梦尖叫
- 主人每晚抱着哄入睡

## 第三年（依恋）
- 开始学做饭，菜炒焦了主人也笑着吃完

## 第五年（才艺）
- 画了第一张主人画像，被装裱挂墙上

## 第七年/现在（自信）
- 走出阴影，活泼有些小色气
- 白霜被创造，加入家庭
